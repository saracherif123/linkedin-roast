# Repo-wide instructions

These rules apply to every skill in this workspace.

## Voice and style

- No emojis.
- No hashtags.
- No AI tells ("In today's fast-paced world", "I'm passionate about", "thrilled to announce").
- Confident but human. Technical when the topic calls for it.
- Short paragraphs. Hook in line 1 for posts.

## LinkedIn brand principles

1. Fix the profile before you post — the funnel starts above the fold.
2. Pick three content lanes. Stay in them for 90 days.
3. Comments and DMs compound faster than posts.
4. Roast with constraints. Rewrite with proof. Multiply with prompts.
5. Ship one rewrite this week. Done > perfect.

## File conventions

- `profile/current.md` — source of truth for the live profile text.
- `voice-samples/` — real writing by the user. Skills must read these before drafting.
- `posts-draft/` — work in progress. Never treat as shipped.
- `posts-shipped/` — published posts for reference and voice training.

## When rewriting profiles

- Headline: outcome, not title. Include "I help [audience] do [outcome]" when possible.
- About: hook in first 3 lines, proof in the middle, one CTA at the end.
- Top Skills: must match positioning, not leftover course artifacts.
- Cut identity clutter. Three identities max.
